# sellingpartner-api-aa-java
## 这是生成的Amazon spapi依赖
使用说明:
1. 声明下载地址
```java
<repositories>
		<repository>
			<id>luodapao</id>
			<name>luodapao</name>
			<url>https://raw.github.com/SnakeBabaluo/sellingpartner-api-aa-java/master/</url>
		</repository>
   </repositories>
   ```
  2. 导入maven下载
  ```java
  <dependency>
      <!--要与打包的项目一样-->
			<groupId>com.amazon.sellingpartnerapi</groupId>
			<artifactId>sellingpartnerapi-aa-java</artifactId>
			<version>1.0</version>
		</dependency>
   ```
